    About This Page

    Component: Stopwatch with Lap Tracking
    A stopwatch that tracks elapsed time and records lap times. Uses an array (laps[]) to store all lap data. Features start, stop, reset, and lap recording buttons.

    Array Usage:
    The laps array dynamically stores each recorded lap time, demonstrating data-driven DOM manipulation.

    jQuery Effects:
    - fadeIn(400) - Smooth fade-in animation when adding new lap times
    - fadeOut(200) then fadeIn(200) - Animated clearing of the lap list on reset
    - jQuery selectors $("#laps") and $("<li>") for efficient DOM manipulation

    Time-Based Effect:
    Uses setInterval() to update the stopwatch display every second while the timer is running.

    Event Listeners:
    Four click event listeners (start, stop, reset, lap) plus keyboard event listener for accessibility (Space/Enter keys).

    Named Functions:
    start(), stop(), reset(), addLap(), and timeToString() handle all stopwatch logic and rendering.

    Accessibility:
    Uses semantic <button> elements, ARIA labels (aria-live="polite"), and keyboard navigation support for inclusive user experience.
